How K-Means Works

K-Means is an iterative clustering algorithm that partitions n data points into K groups based on feature similarity using Euclidean distance minimization.

STEP 01
🎯

Initialize Centroids

Randomly place K centroids in the feature space. K-Means++ initialization picks centroids that are spread apart, reducing convergence time.

STEP 02
📐

Assign Data Points

Each data point is assigned to the nearest centroid using Euclidean distance. This forms K clusters of customers with similar attributes.

STEP 03
🔄

Recompute Centroids

Recalculate each centroid as the mean of all points assigned to its cluster. Centroids shift toward the true center of each group.

STEP 04
✅

Convergence Check

Repeat steps 2–3 until centroids stop moving significantly or the max iterations limit is reached.

RFM Feature Engineering

Before clustering, raw transactional data is transformed into meaningful features. RFM analysis is the industry-standard approach:

  • Recency — Days since the customer's last purchase
  • Frequency — Total number of transactions
  • Monetary — Total amount spent by the customer
  • StandardScaler normalizes features before clustering
  • Elbow Method identifies the optimal K
# Python K-Means Snippet from sklearn.cluster import KMeans from sklearn.preprocessing import StandardScaler scaler = StandardScaler() X_scaled = scaler.fit_transform(rfm) km = KMeans(n_clusters=4, init='k-means++') rfm['Cluster'] = km.fit_predict(X_scaled)
